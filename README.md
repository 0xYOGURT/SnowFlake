# SnowFlake - Particle & Shader Playground

A simple **OpenGL Particle System and Shader Playground** showcasing basic graphics programming techniques.  
This project is designed for beginners to intermediate level, demonstrating:

- Particle systems (fire, smoke, rain)  
- Shader effects (color, gradients, plasma)  
- User interaction (keyboard + mouse controls)  
- Zoom, emitter movement, and live shader switching  

---

## **Features**

- **Particles**
  - Fire, smoke, and rain with velocity, gravity, and random spread
  - Smooth motion and reset on lifespan end

- **Shaders**
  - Gradient, Mandelbrot, Julia fractals, and plasma patterns
  - Real-time switching using keyboard keys

- **Controls**
  | Key / Action | Function |
  |--------------|---------|
  | Arrow Keys   | Move particle emitter |
  | Mouse Scroll | Zoom in/out |
  | `1`          | Fire particle mode |
  | `2`          | Smoke particle mode |
  | `3`          | Rain particle mode |
  | ESC          | Quit program |
